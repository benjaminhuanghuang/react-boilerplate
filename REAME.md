# React + WebPack Project Boilerplate

## Setup
setup.sh

## Key points
- ES6, React

- CSS / SCSS

- Static files (HTML, Images, Font, Icon ...)

- Dev server
    - Hot loading
    - Proxy

- Webpack config for multi environments


## Update packages
```
    npx npm-check-updates -u
```

## Folder Structure
```
assets/
public/
src/

.babelrc
package.json
webpack.config.js
```

## Run project
```
    "start": "webpack serve --env development",
    "build": "webpack --env development"
```
