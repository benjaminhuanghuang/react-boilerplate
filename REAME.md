# React + WebPack Project Boilerplate

## Key points
- Babel
    - ES6,7
    - React

- CSS / SCSS

- Static files (HTML, Images, Font, Icon ...)

- Dev server
    - Hot loading
    - Proxy

- Webpack config for multi environments

## Folder Structure
```
assets/
public/
src/

babel.config.js
package.json
webpack.config.js
```

## Run project
```
    "start": "webpack-dev-server --open --port 3721",
    "build": "webpack"
```
